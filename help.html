הוראות הטמעה

הפונטים המוצגים באתר הם פונטים עבריים וחופשיים לשימוש שרוכזו ונהפכו זמינים להטמעה כפונטי רשת באתרכם. על מנת להטמיע פונטים מהפרויקט באתרכם עקבו אחרי ההוראות הבאות:

כיצד להשתמש בפונט?

1

הוסיפו את הקישור מעמוד הפונט לאתרכם, בין תגיות ה<head>

<link href='http://serve.fontsproject.com/css?family=Nehama:400' rel ='stylesheet' type='text/css'>
2

בגיליון העיצוב (CSS) הוסיפו את משפחת הפונט לסגנונות הרצויים:

#content p { font-family: 'Nehama', serif; }

*מומלץ להוסיף פונט ברירת מחדל לדוגמה סריף או סנס סריף במידה והפונט המבוקש לא זמין.

דוגמה לשימוש

1

העתיקו את הקוד הבא לקובץ HTML

<html> <head> <meta charset="utf-8"> <link href='http://serve.fontsproject.com/css?family=Nehama:400' rel ='stylesheet' type='text/css'> <style> body { font-family: 'Nehama', serif; font-size:48px; } </style> </head> <body> אינטרנט ישראלי זה משהו! </body> </html>
2

פיתחו את הקובץ באמצעות כל דפדפן מודרני. המשפט הבא אמור להיות מוצג בפונט "נחמה"

אינטרנט ישראלי זה משהו!

המשפט מורכב מטקסט רגיל, כך שניתן לשנות את המראה שלו באמצעות הCSS

3

נסו להוסיף צל לסטייל מהדוגמה למעלה

body { font-family: 'Nehama', serif; font-size:48px; text-shadow:4px 4px 4px #aaa; }

כעת המשפט אמור להופיע עם צל מתחתיו

אינטרנט ישראלי זה משהו!

זוהי רק דוגמה קטנה לתוצאות שניתן להגיע אליהן תוך שימוש בפונטים מהאתר ועיצובם בCSS.
